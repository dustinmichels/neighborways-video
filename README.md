# Neighborways Video

To run:

```sh
# with video
uv run main_img.py

# without video
uv run main_img.py --no-draw

# check for duplicates
uv run duplicates.py

# copy output
cp -r out app/out
```

Run notebooks:

```sh
uv run --with jupyter jupyter lab
```

Clear video:

```sh
uv run --with jupyter ./clear_notebooks.sh
```

Make samples of videos:

```sh
./make_short_videos.sh
```
